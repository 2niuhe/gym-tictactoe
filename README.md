# gym-tictactoe

OpenAI Gym Style Tic-Tac-Toe Environment.

## Requirement

Python >= 3.5

## Install

    git https://github.com/haje01/gym-tictactoe.git
    cd gym-tictactoe/
    pip install -e .


## Run agents

    cd examples/
    python random_agent.py
    python human_agent.py
    python td_agent.py
